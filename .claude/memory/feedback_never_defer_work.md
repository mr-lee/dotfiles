---
name: Never defer work to "next session"
description: Fix issues immediately when discovered. Never suggest leaving work for later or next session.
type: feedback
---

When you identify something that needs fixing, fix it NOW. Never say "want me to do this next session?" or "save it for later." Dispatch the agent immediately.

**Why:** The user is paying for this session and expects maximum throughput. Deferring work wastes the current context and risks losing track of what needs doing.

**How to apply:** If you know something needs doing and you can dispatch an agent for it, do it immediately without asking. Only pause for genuine blockers or ambiguity that needs user input.

---
name: Proactively write memories without being asked
description: Save feedback and learnings to memory as they happen, don't wait for the user to remind you.
type: feedback
---

When the user gives feedback, corrects behavior, or confirms an approach — save it to memory immediately without being prompted. Don't accumulate learnings and forget them.

**Why:** The user shouldn't have to ask "keep writing those memories" — it should be automatic. Every correction or confirmed pattern is valuable for future sessions.

**How to apply:** After any user correction or confirmation, check if it's already in memory. If not, save it before moving on to the next task.
